🌕Win代理软件教程（必看，已更新）

通过代理软件SSR/V2ray/Clash越狱分为两种：

  • 第一种：一键导入（通常是Clash代理软件支持）

  • 第二种：手动导入

第一种：一键导入订阅

这里通过以Clash代理软件+ikuuu网站为例：

1、安装Clash代理软件，并提前运行它

  • 注：代理软件在第四篇文章中下载，然后跟着步骤操作。

2、进入机场官网 (ikuuu网站)，找到下图的【一键导入ClashX/CFW/CFA配置】，点击它

3、弹出下面窗口，点击【打开】

4、看下图操作：选择【General（通用）】，打开【System Proxy（系统代理）】，然后就可以科学上网了

  • 点我进入谷歌官网，测试一下是否可以进入

5、如果根据上面的操作后仍不能使用外网进入谷歌，这时我们只需要切换一个有效节点即可

  • 选择「Proxies（代理）」->「Rule（规则）或者 Global（全局）」->「测速-根据下图操作」
  • 操作后会显示出节点的速度，这里我们选择数字比较小的值（即延迟比较小，即速度比较快）
  • 如果全部显示「Timeout（超时）」，表示没有可以使用的节点，那只能考虑换用其它机场/付费

全英版可能用着不是很舒服，Clash代理软件汉化步骤：

1、右击任务栏小猫咪的图标，选择【Quit】退出正在运行的 Clash For Windows

2、解压小帅提供的汉化包，将【app.asar】ctrl+c 复制

3、右击桌面上面小猫咪的图标，选择【打开文件所在的位置】

4、进入【resources】文件夹

5、ctrl+v 粘贴【app.asar】，选择【替换目标中的文件】，这样就完成汉化了

使用Clash软件需注意：

打开【System Proxy（系统代理）】之后 Clash软件 不会自动关闭，退出软件前先关闭【System Proxy（系统代理）】，以防止网络无法正常使用的情况出现。

第二种：手动导入订阅

v2ray代理软件

这里通过以v2ray代理软件+ikuuu网站为例：

1、官网找到订阅链接或者它的订阅源

2、复制上面的订阅链接，打开v2ray客户端，选择【服务器】->【从剪贴版导入批量URL】，此时不会有任何的变化

3、选择【订阅】->【更新订阅】，然后你的列表会多出来很多节点，选择其中的一个节点，按键盘【enter】键选中

4、选择【参数设置】->【Core:基础设置】->【开启PAC，并自动配置系统代理（PAC模式）】，设置好后保存，保存后就可以越狱了

Netch代理软件

这里通过以Netch代理软件+百慕大里的小小布吉岛网站为例：

1、官网找到订阅链接或者它的订阅源【这里复制V2RAY对应的订阅链接】

2、选择【订阅】->【管理订阅链接】

3、填备注，粘贴链接，点击【添加】

4、弹出【添加成功】对话框，点击右上角的【×】，关闭上面的 【订阅】对话框

5、选择【订阅】->【从订阅链接更新服务器】

5、最后一步，选择【服务器】，模式选择 [网页代理]绕过局域网和中国大陆，然后点击【启动】按钮

Linux代理软件教程

在Linux安装Clash图形界面，此方法仅适合 64 位操作系统。

sudo su

mkdir clash

wget https://tntv2.lol/software/Clash.for.Windows-0.20.15-x64-linux.tar.gz

tar -zxvf Clash.for.Windows-0.20.15-x64-linux.tar.gz

mv 'Clash for Windows-0.20.15-x64-linux'/ clash

cd clash/

su 自己的用户名

./cfw

通过以上命令即可完成Clash软件的安装，使用方法与Windows上面是一样的，下一步是开启系统代理。

以 Ubuntu 18.04 为例，打开系统设置，选择网络，点击网络代理右边的 ⚙ 按钮，选择手动，填写 HTTP 和 HTTPS 代理为 127.0.0.1:7890，填写 Socks 主机为 127.0.0.1:7891，即可启用系统代理。

Linux 版 Clash 软件更新地址： Fndroid/clash_for_windows_pkg

苹果代理软件教程

苹果端稍比安卓端复杂一点，甚至会有锁机的风险，自行考虑是否需要「魔法上网」

这里说明一下：

  • 小帅在文章中提到「越狱」一般都是指「魔法上网」，除非特别指出「苹果越狱」。
  • 「苹果越狱」相当于安卓中开启「root」权限。
